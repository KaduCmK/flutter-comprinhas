# Ideias Refinadas

Esta pasta guarda ideias que sobreviveram ao `Inbox` e merecem evolução.

Crie um arquivo por ideia quando houver contexto suficiente para responder:
- por que isso importa
- para quem isso melhora o produto ou a operação
- qual é a próxima etapa concreta

Quando usar:
- a ideia é maior que uma nota de backlog
- existe mais de uma opção possível
- você quer preservar raciocínio, não só a tarefa

Nome sugerido de arquivo:
- `tela-detalhes-produto.md`
- `idempotencia-envio-nfce.md`
- `observabilidade-edge-functions.md`

Template:

```md
# Título

## Contexto

## Problema

## Hipótese

## Impacto esperado

## Restrições

## Próxima ação

## Referências
```
